API: Agendamento de Consultas médicas

Para colocar em prática tudo o que aprendi no meu estágio, desenvolvi uma API RESTful de agendamento de consultas, criando um CRUD completo para gerenciar pacientes, médicos e horários.

O que aprendi no processo:

  • Transformar a arquitetura MVC em API e organizar melhor o código.
  • Identificar e resolver erros com mais rapidez, sem perder tempo preso em problemas.
  • Usar Git e GitHub de forma mais eficiente, seguindo boas práticas de versionamento.
  • Gerar e restaurar scripts SQL para facilitar testes e manter os dados organizados.
  • Documentar a API com Swagger, deixando tudo mais claro e acessível.
  • Implementar autenticação e autorização com JWT, garantindo a segurança dos dados.

    • Os desafios (e o que aprendi com eles)

    Nem tudo foi simples! Tive dificuldades com configuração do Entity Framework Core, injeção de dependência, tratamento de erros e até mesmo com a autenticação e autorização da API. Mas, em vez de desanimar, mergulhei no aprendizado e vi minha evolução a cada obstáculo superado.

    Sistema de avaliação automática de redações usando Processamento de Linguagem Natural (NLP)

    Tenho muito interesse em Inteligência Artificial e Ciência de Dados e, como parte de um projeto acadêmico, desenvolvi um sistema de avaliação automática de redações usando Processamento de Linguagem Natural (NLP) na plataforma Hugging Face.

    A ideia era testar como a IA poderia ajudar na correção de textos. Para isso, utilizei modelos pré-treinados e explorei técnicas de ajuste fino, além de integrar o projeto ao Gradio para criar uma interface simples. No final, consegui um sistema básico que analisa redações e gera uma pontuação automática, com opção para feedback dos usuários.

    Esse projeto me ajudou a entender melhor os desafios da IA, especialmente na interpretação de textos. Ainda tenho muito a aprender, mas essa experiência reforçou meu interesse em continuar estudando e explorando novas formas de aplicar IA e Ciência de Dados na prática.

    Power BI: Dashboards e ETL

    Este foi meu primeiro projeto em Power BI, onde pude aprender e aplicar conceitos básicos de ETL e criação de dashboards. O projeto foi baseado em um storytelling fictício do curso, no qual a dona de um pet shop precisava de insights para expandir seu negócio.

    Durante o desenvolvimento, importei e tratei dados no Power Query, criei colunas calculadas e medidas em DAX, além de explorar diferentes tipos de visualizações. O resultado foi um dashboard interativo que apresenta o faturamento por gênero, bairro e período, com filtros e busca de produtos para facilitar a análise.

    Também aprendi sobre arquitetura MVC, injeção de dependência, persistência de dados com Entity Framework Core, segurança com autenticação JWT e versionamento com Git. Foi uma experiência valiosa para meu crescimento na área.

    Mesmo sendo meu primeiro contato com a ferramenta, essa experiência reforçou meu interesse por dados e BI, além de me aproximar ainda mais das áreas de Inteligência Artificial e Data Science.
